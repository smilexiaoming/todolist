项目结构

初始化npm
```s
npm init -y
```

安装依赖
```s
npm install webpack webpack-cli -D
```

不多说写好webpack配置文件，都2020年了，如果还有童鞋不会基础配置，请官网学习一下，我是不会吐槽你的。
